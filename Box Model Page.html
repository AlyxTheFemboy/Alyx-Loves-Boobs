The Box Model

The box model is often what is referred to when in conversation about CSS layout and design.

It consists of four parts; Margin, Border, Padding and Content

What are each of these elements exactly?

Margin:

The margin is a clear area outside of the border element, this usually helps with website readability and just provides an overall cleaner look.

Border:

The border surrounds the padding and content.

Padding:

The padding is a clear area around the content which is transparent, this also helps with overall readability and provides a clean look.

Content:

The content is basically what it says on the tin, it's the images, text and video embeds etc which is usually the main reason for the website's existence.
